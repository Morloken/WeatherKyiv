A simple weather app for Kyiv, Ukraine. Built with React and Vite.

## What it does

Shows current weather in Kyiv with a nice glassmorphism UI. Uses Open-Meteo API (free, no keys needed).

## Run locally

```bash
npm install
npm run dev:all
```

Open `http://localhost:5173` in your browser.

## Deploy

Push to GitHub and deploy to [Vercel](https://vercel.com) - it's free and works out of the box.

See `DEPLOYMENT.md` if you need help.

## Tech stack

- React + Vite
- Express (local) / Vercel Functions (production)
- Open-Meteo API
- CSS with glassmorphism effects
